tidy data
Tabular data that satisfies four conditions: 1. Each column contains one statistical variable (i.e., one property that was measured or observed). 2. Each different observation is in a different row. 3. There is one table for each set of observations. 4. If there are multiple tables, each table has a column containing a unique key so that related data can be linked.
